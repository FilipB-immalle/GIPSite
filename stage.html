Dagverslagen Stage

30-09-2019

Voor de eerste dag van de observatieweek heb ik een kleine introductie gekregen over WordPress via TeamViewer. Ik heb brongegevens gekopieerd vanuit HTML en CSS code. Daarna heb ik extra pagina's aangemaakt op de website van het bedrijf 2beIT. Ik heb het websitebeheer kunnen zien vanuit het standpunt van de administrator. Verder kreeg ik nog uitleg over hoe webbeheer werkt in bedrijven.

01-10-2019

Ik heb deze dag de website WATOffice geïntegreerd met de kalender van Outlook om verschillende boekingen automatisch te laten registreren in de kalender via een formulier. Daarna heb ik leren werken met verschillende plugins op WordPress (de werking, variëteiten, ...). Ik heb ook nog een ReCAPTCHA ingevoegd op verschillende sites (kimvanreusel, watoffice, 2beit, easyhouse). Dit zorgt voor een beveiliging tegen bots.

02-10-2019

Op de derde dag heb ik tutorials gekeken over hoe je moet werken met CSS en HTML en daarna zelf enkele testjes gedaan. Ik kreeg ook uitleg over sidebars, .php functies, headers, opmaak, ... Ik heb ook verdergewerkt aan de 'Internet Of Things' op de 2beIT webpagina. Ik heb Calendly geïntegreerd in de website van WATOffice als alternatief voor plann3r.

03-10-2019

Ik heb pop-ups leren invoegen op links. Ik heb Calendly aangemaakt voor boekingen voor WATOffice. Ik heb ook uitleg gegeven over WordPress voor Peter over zijn website (Risschot). Ik heb ook een interview gehad met de baas van het bedrijf over zijn onderneming en ook met een werknemer over zijn functie in het bedrijf.

04-10-2019

Op de laatste dag heb ik een kleine webpagina nog kunnen maken voor het bestellen van broodjes voor vergaderingen van de onderneming. Ik heb ook wat meer uitleg gegeven over WordPress voor Peter. 2beIT Heb ik afgewerkt op die dag en verder heb ik nog knoppen verandert (pop-ups) op WATOffice. Daarna heb ik veiligheid toegevoegd op de website 'Xavierleclef'. Het was een aangename ervaring.

07-09-2019

Geachte heer/mevrouw,

Sinds 2018 volg ik de richting IT aan het Immaculata-Instituut. In het 6e en laatste jaar moeten de leerlingen een stage lopen om praktijkervaring in het vakgebied op te doen. Graag zou ik met u een afspraak willen maken om te praten over de mogelijkheid om binnen uw bedrijf een stage te lopen voor de week van 30 september tot 4 oktober. De werkzaamheden bij 2beIT sluiten in mijn ogen goed aan bij mijn interesses en studierichting. Tijdens mijn stage zou ik graag enkel IT-taken willen uitvoeren en natuurlijk zoveel mogelijk bijleren. Ik zie er in ieder geval erg naar uit om mijn kennis die ik de afgelopen jaren heb verworven in de praktijk toe te passen. Ik hoop binnenkort graag van u te horen. Indien u nog vragen heeft aangaande deze brief dan kunt u mij bellen of een mail terugsturen!
